# 🌍 Earth Explorer – Kids Edition

A single HTML file that runs an interactive 3-D Earth in any modern browser.  
Designed for children aged 7–10 to learn geography through play.

## ✨ Features
- Click anywhere on the globe to hear the country name, capital city, and population.
- Speech synthesis reads facts out loud.
- Mini game: “Find France”, “Find Brazil”, etc.
- Toggle night-time city lights and clouds with emoji buttons.
- Works on phones, tablets, laptops, and smart-boards.
- Entirely client-side – no sign-ups, no cost, no tracking.

## 🚀 How to use
1. **Open the page**  
   Simply double-click `index.html` or visit the GitHub Pages link.
2. **Explore**  
   - Drag to spin the globe.  
   - Scroll or pinch to zoom.  
   - Click any land or ocean to get a fun fact.
3. **Play**  
   Press 🎲 “Find a Country” and follow the voice instructions.

## 🛠️ How to host (teachers & parents)
1. Fork or download this repository.
2. Push the file to a new GitHub repo named `earth4kids`.
3. Go to **Settings → Pages** and set the source to the `main` branch.
4. Share the link: `https://<your-username>.github.io/earth4kids/`.

That’s it – the globe is now live for every kid in the world!

## 🧑‍🏫 Tips for educators
- Project the page on a whiteboard and let students take turns finding countries.  
- Encourage them to keep the sound on so everyone hears the spoken facts.  
- Extend the game list by editing the `gameCountries` array at the bottom of the file.
